# React Router v7 Playground

A React Router v7 SPA demonstrating client-side routing with Fluenti.

## Running

```bash
# From monorepo root
pnpm playground:react-router

# Or directly
cd examples/react-router
pnpm dev
```

Dev server starts at `http://localhost:5176`.

## Features Demonstrated

- Client-side routing with React Router v7
- Compile-time `` t`...` `` tagged template translations
- Lazy route loading with `React.lazy()`
- Cookie-based locale persistence
- Locale preloading on hover
